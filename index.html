⚡ Настройка Firebase

Чтобы приложение работало на всех устройствах, нужно подключить Firebase. Это займёт 3 минуты и бесплатно.

1. Создай проект Зайди на console.firebase.google.com → "Создать проект" → введи название (напр. "swift-delivery")
2. Добавь веб-приложение В проекте нажми ⚙️ → "Настройки" → "Твои приложения" → выбери </> (веб) → скопируй конфиг
3. Включи Firestore В меню слева: "Firestore Database" → "Создать базу данных" → выбери "Тестовый режим"
